# Hodl the Coin v1 #

Blockain Game built for Steemit and Ardor.

## Technology
* ESLint
* JavaScript ES6 with Babel
* Webpack
* Firebase (Cloud Firestore)
* Phaser Engine
* Service Workers (offline-plugin)

## Usage

* `yarn install`
* `yarn run serve` or `yarn run serve:ardor`
